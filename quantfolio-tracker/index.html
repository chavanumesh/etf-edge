🔄 Update from Google Sheets

Upload QuantFolio_Portfolio_Tracker.xlsx to Google Drive → open it → File → Save as Google Sheets. The Prices tab already lists every ticker — just wrap each in =GOOGLEFINANCE("NSE:TICKER") (Indian) or =GOOGLEFINANCE("TICKER") (US) to auto-fetch live prices, and set USD_INR_FX using =GOOGLEFINANCE("CURRENCY:USDINR").

➕➖ Full sync (add/remove holdings)
Prices only — publish your Prices tab (File → Share → Publish to web → CSV) with columns Ticker, Price (plus a USDINR row for FX). This updates values on existing holdings but won't add or remove rows.
Full sync — publish your Raw_Data tab as CSV, with columns Ticker, Name, Exchange, Instrument Type, Quantity, Avg Cost, Current Price, Currency. Every refresh will fully rebuild the holdings list from this sheet — add a new row to buy something, delete a row to remove it, and it reflects here automatically.
